# Boards Folder

Recommended location for exported board JSON files.

You can store one board or many boards here.

Local URL pattern:

- `http://127.0.0.1:8000/?board=data/boards/my-board.json&mode=view`

GitHub Pages URL pattern:

- `https://<username>.github.io/<repo>/?board=data/boards/my-board.json&mode=view`
